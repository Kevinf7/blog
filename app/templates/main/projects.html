Property Extractor

propertyextractor.com.au - Lead developer

Utilises the domain.com.au API to search for rental properties in Australia in real-time and to have the results available as a downloadable Excel (csv) file. This website was built for a non-profit org. Users previously spent alot of time copy and pasting search results. This solution provides all search result information in Excel that they can download. The website was built using Python/Flask.

www.arisa.com.au - Lead developer

A student lead generation website built on Flask and incorporating a Google Dialogflow chatbot. Responses are saved into a Mysql database where agencies can then view questions and responses via the backend. Users are also emailed a copy of recommended education institutes and courses. A web scraper using Python / Selenium has also been developed which crawls the CRICOS wesite.
